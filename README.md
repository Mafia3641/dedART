# dedART Editor

Современный текстовый редактор на PyQt6 с профессиональной архитектурой.

## Описание

dedART Editor - это полнофункциональный текстовый редактор, созданный с использованием PyQt6. Приложение предоставляет современный интерфейс с поддержкой различных функций редактирования, подсветки синтаксиса и настраиваемой конфигурации.

## Возможности

- ✅ Современный интерфейс на PyQt6
- ✅ Поддержка открытия и сохранения файлов
- ✅ Темная цветовая схема
- ✅ Номера строк
- ✅ Статусная панель с информацией о позиции курсора
- ✅ Панель инструментов
- ✅ Полнофункциональное меню
- ✅ Система логирования
- ✅ Настраиваемая конфигурация
- ✅ Автосохранение настроек

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd dedART
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
```

3. Активируйте виртуальное окружение:
   - Windows:
   ```bash
   venv\Scripts\activate
   ```
   - Linux/Mac:
   ```bash
   source venv/bin/activate
   ```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование

Запустите редактор:
```bash
python main.py
```

### Основные функции

- **Файл → Новый** (Ctrl+N) - Создать новый файл
- **Файл → Открыть** (Ctrl+O) - Открыть существующий файл
- **Файл → Сохранить** (Ctrl+S) - Сохранить текущий файл
- **Файл → Сохранить как** (Ctrl+Shift+S) - Сохранить файл с новым именем
- **Правка → Отменить** (Ctrl+Z) - Отменить последнее действие
- **Правка → Повторить** (Ctrl+Y) - Повторить отмененное действие

## Архитектура

Проект построен с использованием модульной архитектуры:

- **src/app.py** - Главное окно приложения, управляющее всеми компонентами
- **src/ui/** - Компоненты пользовательского интерфейса (меню, панели)
- **src/editor/** - Компоненты текстового редактора
- **src/utils/** - Вспомогательные модули (логирование, конфигурация)

## Конфигурация

Приложение автоматически создает файл `config.ini` с настройками по умолчанию. Основные секции:

- **General** - Общие настройки (тема, язык, автосохранение)
- **Editor** - Настройки редактора (шрифт, размер табуляции)
- **Interface** - Настройки интерфейса (видимость панелей)
- **Files** - Настройки работы с файлами (кодировка)

## Логирование

Приложение ведет подробные логи в директории `logs/`. Логи включают:
- Информацию о запуске/закрытии приложения
- Операции с файлами
- Ошибки и предупреждения
- Действия пользователя

## Разработка

### Добавление новых функций

1. **Новые действия в меню**: Отредактируйте `src/ui/menu_bar.py`
2. **Новые панели**: Создайте новые классы в `src/ui/`
3. **Новые возможности редактора**: Расширьте `src/editor/text_editor.py`
4. **Новые настройки**: Добавьте в `src/utils/config.py`

### Стиль кода

- Используйте типизацию Python
- Следуйте PEP 8
- Добавляйте документацию к классам и методам
- Используйте логирование для отладки
