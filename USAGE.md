# Инструкция по использованию dedART Editor

## Быстрый старт

1. **Запуск приложения:**
   ```bash
   python main.py
   ```

2. **Открытие файла:**
   - Используйте `Ctrl+O` или меню "Файл → Открыть"
   - Выберите файл `test_file.py` для демонстрации

3. **Редактирование:**
   - Введите текст в редактор
   - Используйте стандартные сочетания клавиш (Ctrl+C, Ctrl+V, Ctrl+X)
   - Наблюдайте за обновлением статусной панели

## Основные функции

### Файловые операции
- **Новый файл** (`Ctrl+N`) - Создание нового документа
- **Открыть** (`Ctrl+O`) - Открытие существующего файла
- **Сохранить** (`Ctrl+S`) - Сохранение текущего файла
- **Сохранить как** (`Ctrl+Shift+S`) - Сохранение с новым именем

### Редактирование
- **Отменить** (`Ctrl+Z`) - Отмена последнего действия
- **Повторить** (`Ctrl+Y`) - Повтор отмененного действия
- **Вырезать** (`Ctrl+X`) - Вырезание выделенного текста
- **Копировать** (`Ctrl+C`) - Копирование выделенного текста
- **Вставить** (`Ctrl+V`) - Вставка из буфера обмена

### Навигация
- **Переход к строке** - Используйте статусную панель для отслеживания позиции
- **Прокрутка** - Используйте полосы прокрутки или колесо мыши

## Интерфейс

### Меню
- **Файл** - Операции с файлами
- **Правка** - Команды редактирования
- **Вид** - Настройки отображения
- **Справка** - Информация о программе

### Панель инструментов
Быстрый доступ к основным функциям:
- Новый файл
- Открыть файл
- Сохранить
- Отменить/Повторить
- Вырезать/Копировать/Вставить

### Статусная панель
Отображает:
- Текущее сообщение
- Позицию курсора (строка, колонка)
- Количество строк в документе
- Кодировку файла

## Настройки

Приложение автоматически создает файл `config.ini` с настройками по умолчанию:

### Основные настройки (General)
- `theme` - Тема интерфейса (dark/light)
- `language` - Язык интерфейса
- `auto_save` - Автосохранение
- `auto_save_interval` - Интервал автосохранения (секунды)

### Настройки редактора (Editor)
- `font_family` - Семейство шрифта
- `font_size` - Размер шрифта
- `tab_size` - Размер табуляции
- `line_numbers` - Показ номеров строк
- `word_wrap` - Перенос слов
- `auto_indent` - Автоотступы

### Настройки интерфейса (Interface)
- `toolbar_visible` - Видимость панели инструментов
- `statusbar_visible` - Видимость статусной панели
- `menubar_visible` - Видимость меню
- `window_width` - Ширина окна
- `window_height` - Высота окна

## Логирование

Приложение ведет подробные логи в директории `logs/`:
- Информация о запуске/закрытии
- Операции с файлами
- Ошибки и предупреждения

## Советы по использованию

1. **Работа с большими файлами:**
   - Используйте поиск по тексту
   - Отключайте номера строк для экономии места

2. **Продуктивность:**
   - Запомните основные сочетания клавиш
   - Используйте панель инструментов для быстрого доступа

3. **Настройка под себя:**
   - Отредактируйте `config.ini` для изменения настроек
   - Настройте размер шрифта под ваши предпочтения

## Устранение неполадок

### Приложение не запускается
1. Убедитесь, что установлены все зависимости: `pip install -r requirements.txt`
2. Проверьте, что Python 3.7+ установлен
3. Проверьте логи в директории `logs/`

### Проблемы с отображением
1. Проверьте настройки шрифта в `config.ini`
2. Убедитесь, что шрифт Consolas доступен в системе
3. Попробуйте изменить тему в настройках

### Проблемы с файлами
1. Проверьте права доступа к файлам
2. Убедитесь, что используется правильная кодировка
3. Проверьте логи для получения дополнительной информации
